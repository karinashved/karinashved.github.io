Hugo Blox Builder создан для того, чтобы технические авторы могли сосредоточиться на содержании, а Hugo Blox — на всем остальном.

Встраивайте видео, подкасты, код, формулы LaTeX и даже тесты для студентов!

На этой странице вы найдёте примеры различных типов технического контента, которые можно использовать с Hugo Blox.

Видео

Преподавайте свои курсы, делясь видео с учениками. Выберите один из подходов:

YouTube:

{{< youtube w7Ft2ymGmfc >}}

Bilibili:

{{< bilibili id="BV1WV4y1r7DF" >}}

Видео файл

Видео можно добавить на страницу, поместив файл в assets/media/ или в папку страницы, а затем встроив его с помощью шорткода video:

{{< video src="my_video.mp4" controls="yes" >}}

Подкаст

Можно добавить подкаст или музыку, поместив MP3-файл в папку страницы или в медиатеку, и встроив аудио на страницу с помощью шорткода audio:

{{< audio src="ambient-piano.mp3" >}}

Проверка знаний

Добавьте простую и весёлую самопроверку с помощью шорткода spoiler:

{{< spoiler text="👉 Нажмите, чтобы посмотреть решение" >}}
Вы нашли меня!
{{< /spoiler >}}
